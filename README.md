# CEMQ

A EMQX client-cli to test and benchmark your EMQX clusters.

## Components

- EMQX-cli
- Logger
- Progress Bar

## Commands & Sub-commands

- help (fl)
- cluster (c)
  - check (sc)
    - connection (sc)
    - health (sc)
  - topics (sc)
  - info (sc)
- bench (c)
  - topic (fl)
  - pub (fl)
  - sub (fl)
  - progress (fl)
  - json (fl)
