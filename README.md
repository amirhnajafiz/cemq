# CEMQ

A EMQX client-cli to test and benchmark your EMQX clusters.

## Components

- EMQX-cli
- Config
- Progress Bar

## Commands & Sub-commands

- help (fl)
- config (c)
  - select (sc)
  - list (sc)
  - info (sc)
- cluster (c)
  - connection (sc)
  - health (sc)
- bench (c)
  - topic (fl)
  - pub (fl)
  - sub (fl)
  - progress (fl)
  - json (fl)
